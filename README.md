# Enoch's Yu's Dotfiles

Welcome to my dotfiles. As a proud minimalist, I enjoy configuring
and trying different setups for my best use. As I constantly update
my dotfiles, decided to share the progress as I learn to configure
and get ideas from other public dotfiles repos. Below are encapsulated
list of what is in this repo with links to the config. Have fun configuring!!

- OS: macOS
- Terminal Emulator: [Alacritty](https://github.com/enochyu-official/dotfiles/tree/main/alacritty)
- Text Editor: [Neovim](https://github.com/enochyu-official/dotfiles/tree/main/nvim)
- Terminal Multiplexer: [tmux](https://github.com/enochyu-official/dotfiles/tree/main/tmux)
- Shell: [Zsh](https://github.com/enochyu-official/dotfiles/tree/main/zsh)
- Tiling Window Manager: [AeroSpace](https://github.com/enochyu-official/dotfiles/tree/main/aerospace)
